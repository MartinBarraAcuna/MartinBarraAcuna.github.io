LaTeX for Philosophy

Martin Barra-Acuña

Purpose of this page

The goal of this page is to make LaTeX more accessible for philosophy. In technical branches of philosophy—especially in philosophy of science, logic, metaphysics, and formal epistemology—the use of formal tools is essential. However, anyone who has attempted to write mathematical documents in word processors like Microsoft Word knows how tedious and cumbersome the process can be.
This is particularly true for logicians, who often struggle to format lengthy proofs properly. Word imposes significant limitations when handling specialized notation. For example, Fitch-style natural deduction is not natively supported, forcing users to insert images instead. This approach compromises both the document's aesthetics and readability, and in the worst cases, can cause formatting issues when multiple images are loaded.
Another common challenge in Word is cross-referencing. The software does not automatically number equations, meaning users must reference them manually. If an equation number changes, every related reference in the text must be updated by hand, making the process inefficient and error-prone.
But none of these things happen in LaTeX. LaTeX addresses these issues by providing a dedicated typesetting system for mathematical notation, automatically numbering equations and managing cross-references. It allows for the creation of complex logical proofs, including Fitch-style deductions, through specialized packages, eliminating the need for cumbersome image insertions. This ensures consistent formatting, readability, and efficient document management, making it ideal for the precision required in philosophical works.

If you do not want enumerate sections, write * after 'section'. Same applies to subsection and subsubsection
